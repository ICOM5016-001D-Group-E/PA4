## Overview
This project implements and evaluates local search algorithms on:
- 8-Queens Problem
- 8-Puzzle Problem

Algorithms tested:
- Steepest-Ascent Hill Climbing  
- Random-Restart Hill Climbing  
- Simulated Annealing  

The program runs experiments and outputs:
- Success rates  
- Graphs comparing performance  

---

## Requirements

Make sure you have Python 3 installed.

Install required libraries:

```bash
pip install matplotlib pandas

```
If that does not work:

```bash
pip3 install matplotlib pandas

```
## How to Run
    1. Open a terminal in the project folder

    2. Run the program:

    3. python main.py  (or use python3 if needed)
